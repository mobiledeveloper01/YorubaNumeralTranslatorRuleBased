# YorubaNumeralTranslatorRuleBased
This repository contains the code for the conversion of English Text or Digit into Yoruba Numerals. It contains standalone module for testing and can be integrated into other applications. You are free to used modify and redistribute. However, you are required to cite the following papers:
	Adewole, L.B. (2026): Development of a rule-based Yorùbá numerals to digits translator, ABUAD International Journal of Natural and Applied Sciences (Accepted) 
	Lawrence B. ADEWOLE, Timilehin V. ADEWOLE, Moyosore C. OWOEYE, Fayoke M., MESE, Oluwatoyin B. ABIOLA (2026).  Development of a Rule-Based Arabic Numerals to Yoruba Translator, ABUAD International Journal of Natural and Applied Sciences (Accepted)
